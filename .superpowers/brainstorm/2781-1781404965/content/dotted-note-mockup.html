A안 · 공식 카드
▲ 위에 기존 음표/쉼표/박자 테이블
🔴 붓점이 뭐예요?
점(·) 하나가 음표 뒤에 붙으면 → 원래 길이의 반(½)만큼 더 늘어나요!
붓점 2분음표
+ 점 =
2박 + 1박 = 3박
붓점 4분음표
+ 점 =
1박 + ½박 = 1½박
붓점 8분음표
+ 점 =
½박 + ¼박 = ¾박
