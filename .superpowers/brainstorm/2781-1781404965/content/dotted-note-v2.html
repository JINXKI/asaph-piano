🎵 붓점 섹션 목업 v2 — A안 스타일 + 붓점 리듬 이어지기
▲ 기존 음표·쉼표·박자 테이블 (변경 없음)
🔴 붓점이 뭐예요?
음표 뒤에 ● 점이 붙으면 → 원래 길이의 반(½)만큼 더 늘어나요!
붓점 2분음표
+ =
2박 + 1박 = 3박
붓점 4분음표
+ =
1박 + ½박 = 1½박
붓점 8분음표
+ =
½박 + ¼박 = ¾박
🩵 붓점 뒤에는 짧은 음표가 따라와요
붓점으로 늘어난 만큼을 짧은 음표로 채워주면 박자가 딱 맞아요!
