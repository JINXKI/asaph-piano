🎵 붓점 + 이음줄 섹션 목업 v3
▲ 기존 음표·쉼표·박자 테이블 (변경 없음)
🔴 붓점이 뭐예요?
음표 뒤에 ● 점이 붙으면 → 원래 길이의 반(½)만큼 더 늘어나요!
붓점 2분음표
+ =
2박 + 1박 = 3박
붓점 4분음표
+ =
1박 + ½박 = 1½박
붓점 8분음표
+ =
½박 + ¼박 = ¾박
🩵 이음줄(타이)이 뭐예요?
같은 음표 두 개를 곡선(⌒)으로 이어주면 → 두 박자를 합쳐서 한 번만 눌러요!
4분음표 + 4분음표
(같은 음에 이음줄)
1박 + 1박
= 2박 🎵
2분음표 + 4분음표
(같은 음에 이음줄)
2박 + 1박
= 3박 🎵
⚠️ 이음줄은 같은 음높이끼리만! 두 번째 음은 다시 누르지 않아요 — 그냥 길게 눌러요~
🔍 붓점 vs 이음줄 — 뭐가 달라요?
🔴 붓점
음표 옆에 점● 하나
길이 = 1½배
음표 하나처럼 생겼어요
