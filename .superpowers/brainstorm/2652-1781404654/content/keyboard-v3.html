5선지 ↔ 건반 매핑 v3

노트 크기 수정 + 손가락 번호 개선 + 흰 건반 윤곽선만. 어떤 방식이 좋으세요?

👇 마음에 드는 디자인 클릭
선택됨 ✓
A. 5선지 위=계이름 / 건반 안=계이름+손가락번호
5선지 노트 위에 계이름만 표시. 건반 키 안에 계이름(위)+손가락번호(아래). 깔끔하고 직관적.
── 계이름 / 5선지 ──
선택됨 ✓
B. 5선지 위=계이름 / 손가락번호 별도 행 / 건반 안=계이름
5선지 아래에 "🖐 손가락" 레이블이 붙은 번호 행을 별도로 추가. 손가락 번호임을 즉시 알 수 있음.
── 계이름 / 5선지 ──
선택됨 ✓
C. 5선지 위=계이름, 아래=손가락번호 / 건반 위=번호 안=계이름
5선지 노트 아래에 손가락 번호 행. 건반 키 상단에도 번호, 하단에 계이름 반복. 정보가 이중으로 연결.
── 계이름 / 5선지 / 손가락번호 ──
클릭하거나 터미널에서 의견 주세요.
