📐 C. 공식 테이블
코드 = 근음 + 3도 + 5도. 각 코드를 수식처럼 설명. 코드 품질(장·단·감)과 구성음이 명확하게 보임.
번호근음3도5도=코드성격
I 도 + 미 + 솔 = C 장3화음
ii 레 + 파 + 라 = Dm 단3화음
iii 미 + 솔 + 시 = Em 단3화음
IV 파 + 라 + 도 = F 장3화음
V 솔 + 시 + 레 = G 장3화음
vi 라 + 도 + 미 = Am 단3화음
vii° 시 + 레 + 파 = Bdim 감3화음
선택됨 ✓
